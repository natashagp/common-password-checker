# Common Password Checker

### Step 1

Clone the repository

```shell
git clone https://github.com/natashagp/common-password-checker.git
```

### Step 2

Change the directory to common-password-checker

```shell
cd common-password-checker
```

### Step 3

Run project

```shell
python main.py
```
